# Unciv-mod-frankie

Unciv mod that adds "大西娘AI文明" civilization.

## Features

### 大西娘AI文明

* **Leader**: 西娘
* **Preferred Victory**: Domination
* **Theme**: Production \& Science focused
* **Unique Ability**: +2 Production from all improved tiles

### Unique Unit: 西娘

* **Type**: Ranged unit
* **Range**: 4 tiles
* **Cost**: 1 Production
* **Strength**: 24 (Melee \& Ranged)
* **Special**: Limited to 1 per civilization
* **Upgrades**: 4 promotion levels with increasing strength

## Installation

1. Copy this mod folder to your Unciv mods directory
2. Enable mod in Unciv settings
3. Start a new game with mod enabled

For `more information on modding, see [the official documentation](https://yairm210.github.io/Unciv/Modders/Making-a-new-Civilization/)
